El juego consiste en adivinar un numero de 4 digitos generado aleatoriamente, este numero tan especial no cuenta con ningun termino 
repetido, cada uno es unico.
Para apoyarte con tu tarea, con cada intento obtendras unas cuentas pistas que te ayudaran a descifrar el numero:

<ul>
  <li>El <b>*</b> significa que un digito es correcto y esta en la misma posicion.</li>
  <li>El <b>-</b> indica que un digito si pertenece al numero pero tiene una posicion diferente.</li>
</ul>

En ningun momento se te indicara a que digito corresponde cada pista, eso lo determinaras tu.
